# 2Brains API

## Description
Mentoring Platform

## Setup environment variables

```bash
$ npm run env:copy

# After this open .env file and setup 
# proper configuration variables 
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
